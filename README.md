# APMtoolbox

This toolbox presents the basic prinicples of analytical probabilistic modeling for scanned particle therapy. Detailed information can be found in:

[Analytical probabilistic modeling for radiation therapy treatment planning](http://iopscience.iop.org/article/10.1088/0031-9155/58/16/5401/pdf)

[Efficiency of analytical and sampling-based uncertainty propagation in intensity-modulated proton therapy](http://iopscience.iop.org/article/10.1088/1361-6560/aa6ec5/meta)
